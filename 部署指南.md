## 📋 下一步操作

### 1. 补充图片素材

参考 `assets/图片清单.md`，准备以下图片：
- 个人照片 1 张
- 案例主视觉 3 张
- 案例细节图 12 张（每个案例 4 张）

**图片规格：**
- 个人照片：3:4 竖版，至少 600px 宽
- 案例主图：16:9 横版，至少 1200px 宽
- 细节图：建议 1000px 宽以上

### 2. 预览效果

在浏览器中打开 `portfolio-lori-base.html` 即可预览。

**注意：** 由于图片尚未补充，会显示图片加载失败。这是正常的，补充图片后即可正常显示。

### 3. 部署上线

#### 方式 1：GitHub Pages（免费，推荐）

```bash
# 1. 在 GitHub 创建仓库（例如：lori-portfolio）
# 2. 将文件重命名
mv portfolio-lori-base.html index.html

# 3. 上传到 GitHub
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/lori-portfolio.git
git push -u origin main

# 4. 在 GitHub 仓库设置中启用 Pages
# Settings → Pages → Source 选择 main 分支
# 几分钟后访问：https://你的用户名.github.io/lori-portfolio
```

#### 方式 2：Vercel（免费，自动部署）

1. 注册 [vercel.com](https://vercel.com)
2. 点击 "Import Project"
3. 连接 GitHub 仓库
4. 自动部署，还能绑定自定义域名

#### 方式 3：直接发送

将整个文件夹打包成 ZIP，发给对方，解压后双击 `index.html` 即可查看。

## 🎨 自定义修改

### 修改颜色主题

在 HTML 文件的 `<style>` 部分找到 `:root` 变量：

```css
:root {
  --accent: #9c7c5c;        /* 主题色（棕色） */
  --pop-orange: #e67e22;    /* 强调色（橙色） */
  --pop-green: #2d5a4a;     /* 辅助色（绿色） */
}
```

### 修改文字内容

直接在 HTML 文件中搜索对应文字，修改即可。

### 添加/删除案例

复制 `<section class="case-section">` 整个区块，修改内容和编号。
